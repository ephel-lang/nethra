A short definition

"Dependent type are types with unklike simple types
[...] depend on or vary with values" - Marting Hoffmann

Dependencies everywhere!

  • terms depend on terms aka λ→
  • terms depend on types aka λ2
  • types depend on types aka λω
  • types depend on terms aka λP

    sig Nat : type
    val Nat = rec(X:type).(Unit | X)

    val zero : Nat = fold (inl unit)
    val succ : Nat -> Nat = fun n -> fold (inr n)

    sig add : Nat -> Nat -> Nat
    val add = fun n1 n2 ->
        case (unfold n1)

    val Functor : (type -> type) -> type = fun M ->
        sig struct
            sig map : {a b:type} -> (a -> b) -> M a -> M b
            -{ Functor laws specification }-
            sig `map id = id` : {a:type} ->
                let lhd = map (id {a}) in
                let rhd = id {M a} in
                lhd = rhd
            sig `map (f o g) = (map f) o (map g)` :
